# Resume-ready project description

## Project title
**FoodLens AI — Multimodal Food Recognition, Nutrition and Recipe Assistant**

## Domain
Multimodal AI · Computer Vision · Generative AI · Health-tech

## Description
Built a Streamlit application that analyzes meal images using Gemini multimodal reasoning, converts model output into validated structured data, matches detected foods with USDA FoodData Central, estimates macro-nutrients from portion weights, and generates personalized recipes. Added manual correction, API retry handling, SQLite history, tests, Docker support, and CI automation.

## Skills
Python, Streamlit, Gemini API, multimodal LLMs, computer vision, Pydantic, REST APIs, USDA FoodData Central, Pandas, SQLite, Pytest, Docker, GitHub Actions.
